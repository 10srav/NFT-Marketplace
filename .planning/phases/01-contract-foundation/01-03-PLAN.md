  <action>
Add Dutch auction functionality to the existing AuctionHouse.sol contract (DO NOT create a new file -- extend the same contract).

**Add struct:**
```solidity
struct DutchAuction {
    address nftContract;
    uint256 tokenId;
    address payable seller;
    uint256 startPrice;
    uint256 endPrice;
    uint256 startTime;
    uint256 duration;
    bool sold;
    bool cancelled;
}
```

**Add state:**
- `uint256 private _nextDutchAuctionId;`
- `mapping(uint256 => DutchAuction) public dutchAuctions;`

**Add events:**
```solidity
event DutchAuctionCreated(uint256 indexed auctionId, address indexed nftContract, uint256 indexed tokenId, address seller, uint256 startPrice, uint256 endPrice, uint256 duration);
event DutchAuctionSold(uint256 indexed auctionId, address indexed buyer, uint256 price);
event DutchAuctionCancelled(uint256 indexed auctionId);
```

**Add functions:**

1. `createDutchAuction(address nftContract, uint256 tokenId, uint256 startPrice, uint256 endPrice, uint256 duration) external whenNotPaused returns (uint256)`
   - Require: `startPrice > endPrice` (price must decrease)
   - Require: `endPrice > 0` (floor must be positive)
   - Require: `duration >= MIN_AUCTION_DURATION && duration <= MAX_AUCTION_DURATION`
   - Require: caller owns the NFT
   - Escrow NFT: `IERC721(nftContract).safeTransferFrom(msg.sender, address(this), tokenId)`
   - Store DutchAuction with `startTime = block.timestamp`
   - Emit DutchAuctionCreated, return auctionId

2. `getCurrentPrice(uint256 auctionId) public view returns (uint256)`
   - If `block.timestamp >= a.startTime + a.duration`, return `a.endPrice` (floor)
   - Otherwise: linear decay = `startPrice - ((startPrice - endPrice) * elapsed / duration)`
   - IMPORTANT: Pure computation, no state writes. This is a view function.

3. `buyDutch(uint256 auctionId) external payable nonReentrant whenNotPaused`
   - Require: `!a.sold && !a.cancelled`
   - Get `currentPrice = getCurrentPrice(auctionId)`
   - Require: `msg.value >= currentPrice`
   - Set `a.sold = true` BEFORE external calls (CEI)
   - Refund overpayment: if `msg.value > currentPrice`, transfer `msg.value - currentPrice` back to buyer
   - Transfer NFT to buyer
   - Call `_deductFeesAndPay(a.nftContract, a.tokenId, a.seller, currentPrice)` for royalty + commission
   - Emit DutchAuctionSold

4. `cancelDutchAuction(uint256 auctionId) external`
   - Require: caller == seller, `!a.sold && !a.cancelled`
   - Set `a.cancelled = true`
   - Transfer NFT back to seller
   - Emit DutchAuctionCancelled

IMPORTANT: Reuse the existing `_deductFeesAndPay` internal function from the English auction work (plan 01-02). Do NOT duplicate it.
