  <action>
Extend the existing Marketplace.sol with offer functionality. Add below the existing code sections, maintaining the comment-section style (e.g., `// ──── Offers ────`).

**Add Offer struct:**
```solidity
struct Offer {
    address payable buyer;
    address nftContract;
    uint256 tokenId;
    uint256 amount;
    uint256 expiry;
    bool active;
}
```

**Add state:**
- `uint256 private _nextOfferId;`
- `mapping(uint256 => Offer) public offers;`
- `uint256 public constant DEFAULT_OFFER_DURATION = 7 days;`

**Add events:**
```solidity
event OfferMade(uint256 indexed offerId, address indexed nftContract, uint256 indexed tokenId, address buyer, uint256 amount, uint256 expiry);
event OfferAccepted(uint256 indexed offerId, address indexed buyer, uint256 amount);
event OfferRejected(uint256 indexed offerId);
event OfferCancelled(uint256 indexed offerId);
```

**Add functions:**

1. `makeOffer(address nftContract, uint256 tokenId, uint256 durationSeconds) external payable nonReentrant whenNotPaused returns (uint256)`
   - Require: `msg.value > 0`
   - Calculate duration: `durationSeconds == 0 ? DEFAULT_OFFER_DURATION : durationSeconds`
   - Require: duration >= 1 hour and <= 30 days (sanity bounds)
   - Store Offer with `expiry = block.timestamp + duration`, `active = true`
   - Emit OfferMade, return offerId

2. `acceptOffer(uint256 offerId) external nonReentrant whenNotPaused`
   - Load offer. Require: `o.active`, `block.timestamp <= o.expiry`
   - Require: `IERC721(o.nftContract).ownerOf(o.tokenId) == msg.sender` (caller must be current NFT owner)
   - Set `o.active = false` BEFORE any external calls (CEI)
   - Call `_unlistIfActive(o.nftContract, o.tokenId)` to cancel any fixed-price listing
   - Transfer NFT from seller to buyer: `IERC721(o.nftContract).safeTransferFrom(msg.sender, o.buyer, o.tokenId)` -- NOTE: seller must have approved marketplace for this token
   - Call `_deductFeesAndPay(o.nftContract, o.tokenId, payable(msg.sender), o.amount)` for royalty + commission
   - Emit OfferAccepted

3. `rejectOffer(uint256 offerId) external`
   - Load offer. Require: `o.active`
   - Require: `IERC721(o.nftContract).ownerOf(o.tokenId) == msg.sender` (caller must be current NFT owner)
   - Set `o.active = false`
   - Transfer `o.amount` back to `o.buyer`
   - Emit OfferRejected

4. `cancelOffer(uint256 offerId) external nonReentrant`
   - Load offer. Require: `o.active`
   - Require: `msg.sender == o.buyer`
   - Set `o.active = false`
   - Transfer `o.amount` back to `o.buyer`
   - Emit OfferCancelled

5. Internal `_unlistIfActive(address nftContract, uint256 tokenId)`:
   - Read `activeListingByToken[nftContract][tokenId]`
   - If > 0 (offset by 1 from plan 01-01): `uint256 listingId = stored - 1`
   - If `listings[listingId].active`: set `listings[listingId].active = false`, `_removeActiveListing(listingId)`, `activeListingByToken[nftContract][tokenId] = 0`, emit `ItemUnlisted(listingId)`

6. Add a view function: `function getOffer(uint256 offerId) external view returns (Offer memory)` for frontend convenience.

IMPORTANT: The seller must have approved the marketplace for the NFT before acceptOffer can transfer it. This is the same pattern as the existing listItem flow. The marketplace does NOT escrow the NFT for offers -- only the ETH is escrowed.

IMPORTANT: Be careful with withdrawCommission -- the contract now holds both commission and escrowed offer ETH. Do NOT change withdrawCommission to sweep the entire balance blindly. Actually, the current `withdrawCommission` already does `payable(owner()).transfer(address(this).balance)` which would sweep escrowed ETH too. This must be fixed: track commission separately.

Add a `uint256 private _accumulatedCommission;` state variable. In `_deductFeesAndPay`, after calculating commission, add `_accumulatedCommission += commission;`. Update `withdrawCommission` to:
```solidity
function withdrawCommission() external onlyOwner {
    uint256 amount = _accumulatedCommission;
    require(amount > 0, "No commission to withdraw");
    _accumulatedCommission = 0;
    payable(owner()).transfer(amount);
}
```
