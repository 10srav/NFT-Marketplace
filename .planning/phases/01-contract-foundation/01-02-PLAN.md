  <action>
Create a new AuctionHouse.sol contract with the following structure:

**Imports and inheritance:**
- Import IERC721, IERC2981, IERC165, ReentrancyGuard, Pausable, Ownable from OZ v5.
- Inherit: `contract AuctionHouse is ReentrancyGuard, Pausable, Ownable`
- Constructor: `constructor() Ownable(msg.sender) {}`

**Commission state** (same pattern as Marketplace.sol):
- `uint256 public commissionRate = 250;` (2.5%)
- `uint256 public constant COMMISSION_DENOMINATOR = 10000;`
- `function setCommissionRate(uint256 newRate) external onlyOwner` with max 1000 (10%)

**English Auction struct:**
```solidity
struct EnglishAuction {
    address nftContract;
    uint256 tokenId;
    address payable seller;
    uint256 startTime;
    uint256 endTime;
    uint256 reservePrice;
    address highestBidder;
    uint256 highestBid;
    bool settled;
    bool cancelled;
}
```

**State:**
- `uint256 private _nextEnglishAuctionId;`
- `mapping(uint256 => EnglishAuction) public englishAuctions;`
- `mapping(uint256 => mapping(address => uint256)) public pendingReturns;`
- `uint256 public constant ANTI_SNIPE_WINDOW = 10 minutes;`
- `uint256 public constant ANTI_SNIPE_EXTENSION = 10 minutes;`
- `uint256 public constant MIN_AUCTION_DURATION = 1 hours;`
- `uint256 public constant MAX_AUCTION_DURATION = 30 days;`

**Events:**
```solidity
event EnglishAuctionCreated(uint256 indexed auctionId, address indexed nftContract, uint256 indexed tokenId, address seller, uint256 reservePrice, uint256 endTime);
event BidPlaced(uint256 indexed auctionId, address indexed bidder, uint256 amount, uint256 newEndTime);
event BidWithdrawn(uint256 indexed auctionId, address indexed bidder, uint256 amount);
event EnglishAuctionSettled(uint256 indexed auctionId, address winner, uint256 amount);
event EnglishAuctionCancelled(uint256 indexed auctionId);
```

**Functions:**

1. `createEnglishAuction(address nftContract, uint256 tokenId, uint256 reservePrice, uint256 duration) external whenNotPaused returns (uint256)`
   - Require: `reservePrice > 0`, `duration >= MIN_AUCTION_DURATION && duration <= MAX_AUCTION_DURATION`
   - Require: caller is owner of the NFT via IERC721.ownerOf
   - Transfer NFT from seller to this contract: `IERC721(nftContract).safeTransferFrom(msg.sender, address(this), tokenId)` (escrow)
   - Store auction struct with `startTime = block.timestamp`, `endTime = block.timestamp + duration`
   - Emit EnglishAuctionCreated
   - Return auctionId

2. `placeBid(uint256 auctionId) external payable nonReentrant whenNotPaused`
   - Require: auction exists, not settled, not cancelled
   - Require: `block.timestamp >= a.startTime && block.timestamp < a.endTime`
   - Require: `msg.value > a.highestBid` (strictly greater)
   - Require: `msg.value >= a.reservePrice`
   - Require: `msg.sender != a.seller`
   - Credit previous highest bidder: `pendingReturns[auctionId][a.highestBidder] += a.highestBid`
   - Update highestBidder and highestBid
   - Anti-sniping: if `a.endTime - block.timestamp < ANTI_SNIPE_WINDOW`, then `a.endTime += ANTI_SNIPE_EXTENSION`
   - Emit BidPlaced with new endTime

3. `withdrawBid(uint256 auctionId) external nonReentrant`
   - Read `pendingReturns[auctionId][msg.sender]`
   - Require: amount > 0
   - Zero BEFORE transfer (CEI pattern)
   - Transfer to msg.sender
   - Emit BidWithdrawn

4. `settleAuction(uint256 auctionId) external nonReentrant`
   - Require: `block.timestamp >= a.endTime`, not settled, not cancelled
   - Set `a.settled = true` BEFORE any external calls
   - If highestBidder != address(0): transfer NFT to winner, call `_deductFeesAndPay` for royalty + commission
   - If no bids: transfer NFT back to seller
   - Emit EnglishAuctionSettled

5. `cancelEnglishAuction(uint256 auctionId) external`
   - Require: caller == seller, not settled, not cancelled
   - Require: `a.highestBidder == address(0)` (no bids placed)
   - Set `a.cancelled = true`
   - Transfer NFT back to seller
   - Emit EnglishAuctionCancelled

6. Internal `_deductFeesAndPay(address nftContract, uint256 tokenId, address payable seller, uint256 salePrice)` -- duplicate the same pattern from Marketplace.sol (plan 01-01). Each contract has its own copy per RESEARCH.md recommendation. Uses try/catch on supportsInterface, queries royaltyInfo, guards combined fees, pays royalty receiver, pays seller.

7. Add `onERC721Received` to accept NFT transfers: implement IERC721Receiver so `safeTransferFrom` to this contract works.

8. `pause()` and `unpause()` onlyOwner functions.

IMPORTANT: Every state-changing function that moves ETH or NFTs MUST be `nonReentrant`. Settlement MUST update state BEFORE external calls (CEI pattern).
